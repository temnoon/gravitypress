This is my personal notes file for what I am trying to do with this project, gravitypress. 

I've owned GravityPress.org for many years now, and have used it as a personal notebook archive, but it fell into disrepair, and is now only a stub site, with good intentions. 

The ability to create a versitile Notebook Builder has reignited my hopes for GravityPress. I want to empower people to use pen and paper as a tool for their own mind. This may be our last hope at doing something like that, for this world.

I am currently working hard on getting the PolarGraph Designer to be easy to use to create a Polar Graph editor with many degrees of freedom. That will create image assets, and allow direct creation of svg, png and other file types.

I want to create Designers for lined pages, cartesian graphs, and image pages, that would also include renderable text blocks, masking and transparencies. 

Each of these graphic editors produce an asset which can then be used as a layer, which the notebook editor can composit and arrange in layers with transparencies and combination methods between layers to produce a merged page, which can be attached to a Notebook spread, which the Notebook Builder calculates into what page number could be imprinted on that page based on the total pages in the notebook, when creating a saddle-stitched booklet from letter size (or other size) pages.

This notebook is also an asset which can be saved, recalled, and used by other users of the system. The notebook can be saved as a pdf, or other useful formats. Eventually I want to have members on the website who share their notebook designs, subscribe to each other's notebook posts, mail physical notebooks to each other, post and comment on each other's notebooks, and share their notebooks with the world.

The next step is getting the color gradients to work versitily. I want to be able to create full rainbows, and also have the ability to create gradients with a single color, and then have the ability to create gradients with two colors, and so on.

I want to be able to vary the length of the gradient with a slider, and also the angle of the gradient, and also the opacity of the gradient.

I want to be able to reverse the direction of the gradient. I also want to be able to invert the colors of the gradient (purely mathematically, and reversibly)
